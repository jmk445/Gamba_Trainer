Connect via Bluetooth

The TF4Micro motion kit communicates with this website via BLE so you can have a wireless experience. First, learn specific gestures through Tiny Motion Trainer and download the model. Second, load the saved model with .tflite extension and click the "Connect" button to transfer the model to the ESP32s3 board.

Sending Tensorflow model to your ESP32s3 via Bluetooth

Play

Attach the board to the back of a finger(close to the tip). Make sure the LED is facing you and the USB connector is towards your wrist.

There are 5 gestures :

1. Swipe left
2. Swipe right
3. Twirl (do so quickly!)
4. Pluck (which changes the drum loop)
5. Poke

Check out the gifs for guidance. Have fun!
